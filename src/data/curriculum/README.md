# VibeStudy Curriculum

Учебные программы для каждого языка программирования.

## Доступные курсы

| Язык | Файл | Продолжительность |
|------|------|-------------------|
| Python | [python.md](./python.md) | 30 дней |
| JavaScript | [javascript.md](./javascript.md) | 30 дней |
| TypeScript | [typescript.md](./typescript.md) | 30 дней |
| Go | [go.md](./go.md) | 30 дней |
| Rust | [rust.md](./rust.md) | 30 дней |
| Java | [java.md](./java.md) | 30 дней |
| C++ | [cpp.md](./cpp.md) | 30 дней |
| Swift | [swift.md](./swift.md) | 30 дней |

## Структура курса

Каждый курс разбит на **4 недели**:

1. **Неделя 1**: Основы языка
2. **Неделя 2**: Структуры данных и коллекции  
3. **Неделя 3**: Функции и ООП
4. **Неделя 4**: Продвинутые темы + Финальный проект

## Формат дня

Каждый день включает:
- Теоретический материал
- Примеры кода
- Практические задания
- Мини-проект (каждый 7-й день)

## Использование в приложении

Эти MD файлы используются как источник данных для страницы `/lessons` где пользователи проходят уроки.
